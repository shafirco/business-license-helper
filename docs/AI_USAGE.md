# תיעוד שימוש ב-AI - מערכת הערכת רישוי עסקים

## כלי פיתוח AI שנעשה בהם שימוש

### 1. Cursor AI (כלי הפיתוח הראשי)
**תיאור:** סביבת פיתוח מתקדמת עם יכולות AI מובנות
**שימוש:**
- כתיבה וערכת קוד בזמן אמת
- הצעות השלמות חכמות
- דיבוג ופתרון בעיות
- רפקטורינג אוטומטי

**יתרונות:**
- אינטגרציה חלקה עם VS Code
- הבנה מעולה של קונטקסט הפרויקט
- תמיכה בטכנולוגיות מרובות (React, Node.js, CSS)

### 2. Claude Sonnet 4 (מודל השפה לפיתוח)
**תיאור:** מודל שפה מתקדם המיועד לסיוע בפיתוח
**שימוש:**
- ייצור קוד מורכב
- פתרון בעיות ארכיטקטוניות
- יצירת תיעוד מקיף
- אופטימיזציה של ביצועים

**יתרונות:**
- הבנה עמוקה של דרישות טכניות
- יכולת לייצר קוד איכותי במגוון שפות
- הצעות פתרונות יצירתיות

### 3. OpenAI GPT-3.5-turbo (מודל השפה במוצר)
**תיאור:** מודל השפה המשמש במערכת עצמה ליצירת הדוחות
**שימוש:**
- יצירת דוחות מותאמים אישית
- תרגום "שפת חוק" לשפה עסקית
- סיכום והמלצות חכמות

## מודל השפה מרכזי - OpenAI GPT-3.5-turbo

### מדוע נבחר GPT-3.5-turbo:
1. **איזון עלות-ביצועים** - משמעותית זול יותר מ-GPT-4
2. **מהירות תגובה** - זמן תגובה מהיר למשתמש קצה
3. **תמיכה בעברית** - תוצאות איכותיות בעברית
4. **יציבות** - נמוך יותר בשינויים פתאומיים

### הגדרות המודל:
```javascript
const completion = await openaiInstance.chat.completions.create({
    model: "gpt-3.5-turbo",
    messages: [
        {
            role: "system",
            content: "אתה מומחה ברישוי עסקים בישראל..."
        },
        {
            role: "user", 
            content: prompt
        }
    ],
    temperature: 0.3, // טמפרטורה נמוכה לעקביות
    max_tokens: 1500  // הגבלת אורך לתוצאות ממוקדות
});
```

## Prompts עיקריים שנעשה בהם שימוש

### 1. Prompt למערכת הדוחות
```javascript
const systemPrompt = `אתה מומחה ברישוי עסקים בישראל. צור דוח קצר ומדויק בעברית בלבד. 
הסיכום צריך להיות טקסט רציף ללא מספור ובעברית מלאה. 
שאר הסעיפים עם רשימה ממוספרת (1., 2., וכו'). 
היה קצר וברור. אל תשתמש במילים באנגלית או בשמות באנגלית. הכל בעברית!`;

const userPrompt = `עסק: ${businessType}, גודל: ${businessSize} מ"ר, תכונות: ${features || 'אין'}

דרישות קריטיות:
${requirementsSummary}

צור דוח קצר עם:
- סיכום: טקסט רציף של 2-3 משפטים בעברית בלבד (ללא מספור, ללא מילים באנגלית, ללא מירכאות, תשתמש רק בשמות העסק בעברית)
- דרישות דחופות: רשימה של 3 פריטים מקסימום עם מספור
- המלצות: רשימה של 3 פריטים מקסימום עם מספור

פורמט: הסיכום ללא מספור, שאר הסעיפים עם 1., 2., וכו'. 
אל תכלול צעדים מעשיים או הערות חשובות. הכל בעברית בלבד! השתמש רק בשמות עסקים בעברית!`;
```

### 2. Prompts לפיתוח עם Claude/Cursor

#### יצירת קומפוננטת React:
```
צור קומפוננטת React לטופס שאלון עסק שכולל:
- שדות לסוג עסק, גודל, ומאפיינים
- validation
- עיצוב מודרני עם CSS
- תמיכה בעברית וכיוון RTL
```

#### אופטימיזציה של קוד:
```
בדוק את הקוד הבא ושפר אותו:
1. הוסף error handling
2. שפר את הביצועים  
3. הוסף הערות והסברים
4. ודא שהקוד עוקב אחרי best practices

[קוד מקורי]
```

#### פתרון בעיות:
```
יש לי בעיה עם [תיאור הבעיה].
הקוד הנוכחי:
[קוד]

השגיאה:
[הודעת שגיאה]

איך לתקן את זה?
```

### 3. Prompts ספציפיים לתיקון בעיות

#### תיקון טקסטים קטועים:
```
הפונקציה הבאה חותכת טקסטים באמצע משפטים:
[קוד מקורי]

צור פונקציה משופרת שתשמור על משפטים שלמים ותמנע חיתוך באמצע מילים או משפטים.
```

#### הסרת תווים משפטיים:
```
צור regex patterns להסרה של:
- ציטוטים משפטיים בעברית כמו "התשל\"א - 1971"
- מספרי תקנות כמו "ישראלי 1145"
- הפניות בסוגריים כמו "(א)"
- שנים בסוגריים

הקפד על טיפול נכון בעברית ובתווים מיוחדים.
```

#### עיצוב מתקדם:
```
צור CSS מודרני עם:
- רקע מונפש עם gradient
- אנימציות float ו-pulse
- אפקטי hover אטרקטיביים
- תמיכה מלאה ב-RTL לעברית
- transition effects חלקים
```

### הצלחות מרכזיות:

### אתגרים שהתגברנו עליהם:
